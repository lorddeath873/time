---------------------------
Installation
---------------------------

1. Inhalt aus dem Ordner "time" (time.zip -> time) in Ihr Rootverzeichnis hochladen.
2. http://ihre-website.de/install aufrufen und Anweisungen folgen.

---------------------------
WICHTIG!!! UPDATES !!!
---------------------------
Da bisher kein Updatescript existiert, bitten wir Sie jedesmal ein Backup ihrer
Datenbank zu erstellen.

Kopieren Sie das Update über Ihre existierende Installation und führen Sie in Ihrer Datenbank für JEDE Tabelle
"Truncate" aus.

Nach der Installation spielen sie bitte das Backup Ihrer Datenbank wieder ein.
